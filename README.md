# Rakt Setu

Rakt Setu is an AI-powered blood donation and emergency response system.

## Features
- 🩸 Real-time donor tracking
- 📍 Google Maps integration
- 🤖 WhatsApp AI chatbot (Twilio + Gemini)
- 📊 Demand vs Supply analytics dashboard

## Tech Stack
- Frontend: React
- Backend: Node.js
- Database: Firebase Firestore
- APIs: Google Maps, Twilio, Gemini AI

## Setup
1. Clone the repo
2. Run `npm install`
3. Add `.env` file
4. Run `npm start`
